Reconstruye y devuelve un objeto de tipo \doxylink{a00301}{Usuarios} integrando todas las propiedades almacenadas en la sesión global. 

Este método realiza una operación de ensamblado para convertir las propiedades planas de \doxylink{a00349}{Global\+Data} en una estructura jerárquica compleja. Es fundamental para operaciones de persistencia, permitiendo que otros servicios (como el cliente de base de datos) reciban una entidad completa con sus objetos anidados de \doxylink{a00305}{Perfil\+Usuario}, \doxylink{a00309}{Estadisticas\+Usuario}, \doxylink{a00313}{Listas\+Usuario} y \doxylink{a00317}{Configuracion\+User}. 

\begin{DoxyReturn}{Devuelve}
Una nueva instancia de \doxylink{a00301}{Usuarios} que refleja el estado actual de la sesión del usuario, incluyendo sus preferencias de configuración y listas sociales. 
\end{DoxyReturn}

void Beta\+Proyecto.\+Singleton.\+Global\+Data.\+Set\+User\+Data (\begin{DoxyParamCaption}\item[{\doxymbox{\hyperlink{a00301}{Usuarios}}}]{user}{}\end{DoxyParamCaption})}



Sincroniza y mapea la información completa de un objeto \doxylink{a00301}{Usuarios} hacia las propiedades globales de la sesión actual. 

Este método actúa como un adaptador que distribuye los datos del usuario autenticado en diferentes categorías:\+ 
\begin{DoxyItemize}
\item {\bfseries{Datos de Identidad:\+}} Mapea ID, nombre, correo y rol directamente desde la raíz del objeto. 
\item {\bfseries{Perfil y Preferencias:\+}} Extrae información geográfica, imagen de perfil y estado de privacidad. 
\item {\bfseries{Actividad y Social:\+}} Inicializa contadores de estadísticas y asegura que las listas de seguidores y favoritos no sean nulas. 
\item {\bfseries{Configuración de Entorno:\+}} Carga los diccionarios de tema, idioma y fuente, aplicando valores por defecto si no existen preferencias guardadas. 
\end{DoxyItemize}Se utiliza principalmente durante el inicio de sesión o tras una actualización exitosa del perfil del usuario para mantener la consistencia en toda la aplicación. 


\begin{DoxyParams}{Parámetros}
{\em user} & El objeto \doxylink{a00301}{Usuarios} recuperado de la base de datos que contiene la información maestra.\\
