65 /// </summary>

67 /// notificado para proceder. Si el inicio de sesión falla debido a credenciales incorrectas o problemas de conexión, una alerta es

69 /// actualización del estado global y visualización de alertas. </remarks>
70 /// <returns>Devuelve una tarea que representa la operación de inicio de sesión asíncrono. La tarea se completa cuando el intento de inicio de sesión tiene
71 /// terminado, independientemente del éxito o el fracaso. </returns>
72 private async Task IntentarLogin()
